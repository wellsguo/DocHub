## 解决方案
- 商业软件 ArcGIS
  - 绘图   
    arcgis desktop
  - 发布地图服务  
    arcgis server
  - 页面开发  
    arcgis js api 
- 开源软件
  - 绘图  
    [qgis](http://qgis.org/) / [uDig](http://udig.refractions.net/) / [MapWindow](https://www.mapwindow.org/) / [gVSIG](http://www.gvsig.com/) / [SAGA GIS](http://www.saga-gis.org/en/index.html) / [GRASS GIS](https://grass.osgeo.org/)
  - 地图服务  
    [geoserver]() 
  - 提供地理数据库支持  
    [postgresql & postgis]() 
  - 页面开发
